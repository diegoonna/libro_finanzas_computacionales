# Bucles `while`, Contadores y Acumuladores

La estructura `while` repite las instrucciones mientras una condición sea evaluada como verdadera (`True`)[cite: 30, 31]. Se le conoce como "bucle indefinido" porque no sabemos de antemano cuántas veces se ejecutará[cite: 86].

**Conceptos clave:**

* **Contador:** Una variable que suma un valor constante en cada iteración (ej: `i = i + 1`)[cite: 58].
* **Acumulador:** Una variable que suma un valor variable en cada iteración (muy usado para sumar flujos, capitales, etc.).

```{pyodide-python}
#| label: while-ahorro
# Ejemplo: Búsqueda incremental (Aproximación por tanteo)
# Supongamos que queremos saber cuántos meses necesitamos ahorrar $150 para alcanzar o superar los $1000, 
# asumiendo que no hay interés.

ahorro_mensual = 150
objetivo = 1000

# Inicializamos nuestras variables
capital_acumulado = 0  # Esto es un ACUMULADOR
meses = 0              # Esto es un CONTADOR

print("--- Inicio del ahorro ---")
while capital_acumulado < objetivo:
    capital_acumulado = capital_acumulado + ahorro_mensual
    meses = meses + 1
    print(f"Mes {meses}: Acumulado = ${capital_acumulado}")

print(f"Tomó {meses} meses alcanzar el objetivo de ${objetivo}.")
```

# Bucles `for` y la función `range()`

La estructura `for` permite realizar una cantidad definida de iteraciones[cite: 90]. Itera sobre cada elemento de una secuencia (como una lista o un rango de números)[cite: 103]. La función `range()` se utiliza frecuentemente para construir estos bucles[cite: 172].

```{pyodide-python}
#| label: for-interes-simple
# Ejemplo: Proyección de un capital con interés simple
# Queremos ver cómo crece una inversión inicial de $500 a una tasa del 5% anual durante 4 años.

capital_inicial = 500
tasa_anual = 0.05
anios = 4

print(f"Capital inicial: ${capital_inicial}")

# range(1, anios + 1) generará los números: 1, 2, 3, 4
for anio in range(1, anios + 1):
    # Calculamos el interés generado en el año (interés simple)
    interes_generado = capital_inicial * tasa_anual
    
    # En el interés simple, el capital total es el inicial más los intereses acumulados
    capital_total = capital_inicial + (interes_generado * anio)
    
    print(f"Fin del Año {anio}: El capital total es ${capital_total}")
```

# Listas: Almacenando múltiples valores

Las listas nos permiten guardar varios elementos en una sola variable, manteniendo el orden[cite: 115, 207]. Podemos acceder a ellos mediante su índice (que empieza en 0) [cite: 122, 129] y modificarlas con métodos como `.append()`, `.pop()` o `del`[cite: 216, 218].

```{pyodide-python}
#| label: listas-filtrado
# Ejemplo A: Filtrado de datos en listas
# Tenemos una lista con las cotizaciones históricas de un tipo de cambio.
# Queremos crear una nueva lista SOLO con los días en que la cotización superó los $1000.

cotizaciones = [980.5, 995.0, 1005.2, 1010.0, 998.0, 1020.5]
cotizaciones_altas = [] # Creamos una lista vacía usando corchetes

for precio in cotizaciones:
    if precio > 1000:
        cotizaciones_altas.append(precio) # Añade el elemento al final de la lista [cite: 204, 216]

print("Cotizaciones originales:", cotizaciones)
print("Cotizaciones altas (mayores a 1000):", cotizaciones_altas)
```

```{pyodide-python}
#| label: listas-variaciones
# Ejemplo B: Operaciones secuenciales (diferencias de precios)
# A veces necesitamos comparar un elemento de la lista con el anterior.
precios_accion = [50.0, 52.0, 51.0, 55.0]
variaciones = []

# Iteramos usando índices. Usamos len() - 1 porque compararemos el día actual con el siguiente.
for i in range(len(precios_accion) - 1):
    precio_hoy = precios_accion[i]
    precio_manana = precios_accion[i + 1]
    
    variacion_absoluta = precio_manana - precio_hoy
    variaciones.append(variacion_absoluta)

print("Precios diarios:", precios_accion)
print("Variaciones absolutas diarias:", variaciones)
```

# Diccionarios: Datos etiquetados

A diferencia de las listas, los diccionarios no tienen un orden secuencial (son como una bolsa)[cite: 402]. Los elementos se guardan y se buscan utilizando "claves" (keys) en lugar de índices numéricos[cite: 403, 438].

```{pyodide-python}
#| label: dict-bono
# Ejemplo A: Mapeo de información de un activo
# Creamos un diccionario vacío usando dict() o llaves {}[cite: 404].
bono_soberano = dict()

# Agregamos elementos clave-valor
bono_soberano['ticker'] = 'AL30'
bono_soberano['paridad'] = 45.5
bono_soberano['legislacion'] = 'Local'

print("Datos del bono:", bono_soberano)
print("Ticker específico:", bono_soberano['ticker'])

# Actualizamos un valor existente
bono_soberano['paridad'] = 48.2
print("Paridad actualizada:", bono_soberano['paridad'])
```

```{pyodide-python}
#| label: dict-riesgos
# Ejemplo B: Frecuencias y el método .get()
# Queremos contar cuántos activos tenemos agrupados por su nivel de riesgo.
# El método .get(clave, valor_por_defecto) es ideal para esto[cite: 475, 477].

activos = ["Bajo", "Alto", "Medio", "Bajo", "Bajo", "Medio", "Alto", "Bajo"]
conteo_riesgos = {}

for nivel in activos:
    # Si 'nivel' ya está en el diccionario, toma su valor actual.
    # Si no está, devuelve 0. A eso le sumamos 1.
    conteo_riesgos[nivel] = conteo_riesgos.get(nivel, 0) + 1 # [cite: 481]

print("Distribución de riesgos de la cartera:")
print(conteo_riesgos)
```

```{pyodide-python}
#| label: dict-inventario
# Ejemplo C: Combinando diccionarios (Valuación de inventario)
# Tenemos un inventario físico de commodities y sus precios de mercado.
# Queremos calcular el valor total del inventario.

cantidades = {'Soja': 150, 'Maiz': 300, 'Trigo': 200}
precios_mercado = {'Soja': 350.0, 'Trigo': 210.5, 'Maiz': 180.0}

valor_total_inventario = 0 # Acumulador

# Recorremos las claves del diccionario de cantidades [cite: 508]
for commodity in cantidades:
    cantidad_toneladas = cantidades[commodity]
    precio_tonelada = precios_mercado[commodity]
    
    valor_parcial = cantidad_toneladas * precio_tonelada
    valor_total_inventario = valor_total_inventario + valor_parcial

print(f"El valor total del inventario agrícola es: ${valor_total_inventario}")
```
